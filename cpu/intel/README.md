# Intel 所有CPU产品目录

这中间涉及 **CPU** 相关专业参数，不懂的同学可以自学成才。价格是基于 **强哥**的 **京东** 和 **马爸爸**的 **天猫/淘宝**。

主要技术参数有：

 `核心数`、`线程数`、`基础频率`、`睿频频率`、`功率`、`内存`、`内存通道数`、`PCIE通道数`、`工艺`、`L3缓存`
 
 > 暂时不考虑i3处理器及更低端的处理器。
